\doxysubsection{Detailed Description}
Methods for communcations with the MSP432. 

\mbox{\hyperlink{_serial_8h_source}{Serial.\+h}}

This class uses a com port to emulate serial communcation with the MSP432 using the \mbox{\hyperlink{_serial_8h_source}{Serial.\+h}} library with the OOP 4618 serial protocol
